## البنية التحتية:

### Backend
- Node.js + Express.js
- PostgreSQL Database
- JWT Authentication

### Frontend (Web)
- React.js
- Material-UI / Ant Design
- Redux State Management

### Desktop (Electron)
- Electron Framework
- SQLite Local Database
- Offline Mode Support

## التثبيت والتشغيل

```bash
# تثبيت الحزم
npm install

# تشغيل التطوير
npm run dev

# بناء نسخة الإنتاج
npm run build
```
